# **KSeF 2.0 \- Руководство для Интеграторов**
22.12.2025

Данный документ представляет собой компендиум знаний для разработчиков, аналитиков и интеграторов систем, которые реализуют интеграцию с Национальной системой электронных счетов-фактур (KSeF) версии 2.0. Руководство сосредоточено на технических и практических аспектах, связанных с коммуникацией с API системы KSeF.

##  Введение в KSeF 2.0

Национальная система электронных счетов-фактур (KSeF) — это центральная телеинформационная система, предназначенная для выставления и получения структурированных счетов-фактур в электронном виде.

Для каждой области предусмотрено:

* детальное описание работы,
* пример вызова на языках C# и Java,
* связи со спецификацией [OpenAPI](https://api-test.ksef.mf.gov.pl/docs/v2) и кодом справочной библиотеки.

Примеры кода, представленные в руководстве, подготовлены на основе официальных open source библиотек:
* [ksef-client-csharp](https://github.com/CIRFMF/ksef-client-csharp) – библиотека на языке C#
* [ksef-client-java](https://github.com/CIRFMF/ksef-client-java) – библиотека на языке Java

Обе библиотеки поддерживаются и развиваются командами Министерства финансов и предоставляются публично на условиях open source. Они обеспечивают полную поддержку функциональности API KSeF 2.0, включая обработку всех endpoints, моделей данных и примерных сценариев вызовов. Использование этих библиотек позволяет значительно упростить процесс интеграции и минимизирует риск неправильной интерпретации контрактов API.
