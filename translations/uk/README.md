# **KSeF 2.0 — Посібник для Інтеграторів**
22.12.2025

Цей документ є компендіумом знань для розробників, аналітиків та інтеграторів систем, які реалізують інтеграцію з Національною системою електронних рахунків-фактур (KSeF) версії 2.0. Посібник зосереджується на технічних та практичних аспектах, пов'язаних з комунікацією з API системи KSeF.

## Вступ до KSeF 2.0

Національна система електронних рахунків-фактур (KSeF) — це центральна телеінформаційна система, призначена для виставлення та отримання структурованих рахунків-фактур в електронному вигляді.

Для кожної області передбачено:

* детальний опис роботи,
* приклад виклику мовами C# та Java,
* зв'язки зі специфікацією [OpenAPI](https://api-test.ksef.mf.gov.pl/docs/v2) та кодом еталонної бібліотеки.

Приклади коду, представлені в посібнику, були підготовлені на основі офіційних бібліотек з відкритим вихідним кодом:
* [ksef-client-csharp](https://github.com/CIRFMF/ksef-client-csharp) — бібліотека мовою C#
* [ksef-client-java](https://github.com/CIRFMF/ksef-client-java) — бібліотека мовою Java

Обидві бібліотеки підтримуються та розвиваються командами Міністерства фінансів і надаються публічно на засадах відкритого коду. Вони забезпечують повну підтримку функціональності API KSeF 2.0, включаючи обслуговування всіх endpoints, моделей даних та прикладових сценаріїв викликів. Використання цих бібліотек дозволяє значно спростити процес інтеграції та мінімізує ризик неправильної інтерпретації контрактів API.
